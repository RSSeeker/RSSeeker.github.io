# RSSeeker Web 神经网络训练器

这是一个基于单页静态页面的神经网络训练演示器，支持回归与分类任务、可视化训练曲线、混淆矩阵和 2D 决策边界。

## 功能

- 支持自定义输入/输出维度与隐藏层结构
- 支持多种激活函数：ReLU、Leaky ReLU、Sigmoid、Tanh
- 支持多种优化器：SGD、Momentum、RMSProp、Adam
- 支持 Dropout、L2 正则化、测试集划分
- 支持分类任务的混淆矩阵展示
- 支持输入维度为 2 的分类任务决策边界可视化
- 支持移动端界面适配

## 使用方式

1. 打开 `index.html`
2. 添加训练数据或点击示例按钮加载预设数据
3. 配置网络结构与训练参数
4. 点击“开始训练”进行模型训练
5. 在分类任务中，可查看混淆矩阵与 2D 决策边界

## 文件说明

- `index.html`：主页面与全部前端逻辑
