# 神经网络训练器

一个纯前端、零依赖的神经网络训练演示工具，支持回归与分类任务，内置可视化训练曲线、混淆矩阵和 2D 决策边界。

## 功能特性

- **网络类型**：回归网络（MSE / 线性输出）与分类网络（Softmax + Cross Entropy）
- **网络结构**：自定义输入/输出维度与任意层隐藏层，支持动态增删
- **激活函数**：ReLU / Leaky ReLU / Sigmoid / Tanh
- **优化器**：SGD / Momentum / RMSProp / Adam
- **正则化**：Dropout（可配比率）、L2 权重衰减（可配系数）
- **数据划分**：支持训练集 / 测试集自动划分，可自定义测试比例
- **早停机制**：基于验证 loss 的 Early Stopping
- **GPU 加速**：支持 WebGL 硬件加速开关（控制 Canvas 渲染管线）
- **训练曲线**：Loss 曲线 + 分类准确率双 Y 轴（Chart.js）
- **混淆矩阵**：分类任务自动展示，含每类 Precision / Recall / F1-Score
- **决策边界**：输入维度为 2 的分类任务自动绘制决策边界
- **模型持久化**：JSON 格式导出/导入，包含网络权重、超参数、训练数据与历史曲线
- **推理面板**：独立推理模块，分类任务显示各类别概率
- **预设示例**：XOR、非线性拟合、正弦曲线、鸢尾花分类、螺旋分类、四象限分类
- **移动端适配**：响应式布局，手机/平板可用

## 使用方式

1. 用浏览器直接打开 `index.html`
2. 点击示例按钮加载预设数据（或手动添加训练数据）
3. 配置网络结构与训练参数（学习率、epochs、批大小、激活函数等）
4. 点击「开始训练」观察实时 Loss 曲线
5. 训练完成后查看混淆矩阵、决策边界，在推理面板测试新样本
6. 使用「保存模型」导出 JSON 文件，「加载模型」恢复

## 技术栈

| 组成部分 | 实现 |
|----------|------|
| 神经网络 | 纯 JavaScript 手写，含前向传播、反向传播（BPTT）、He 初始化 |
| 优化器 | 手写 Adam / Momentum / RMSProp / SGD |
| 正则化 | Dropout（伯努利 mask）、L2 权重衰减 |
| 损失函数 | MSE（回归）、Cross Entropy（分类 + Softmax） |
| 图表 | [Chart.js 4](https://www.chartjs.org/)（CDN 引入） |
| 样式 | 纯 CSS，深色主题，响应式布局 |

## 说明

该项目无需安装任何依赖，仅需一个现代浏览器即可运行。适合用于教学、演示和简单模型实验。

所有运算均在浏览器中完成（CPU 或 WebGL 加速），训练数据不会上传到任何服务器。
